Overview
This project streamlines and automates various garage management operations—covering customer information, appointment scheduling, service tracking, billing, and feedback collection—using Salesforce tools.
Key features include:

Custom Objects and Relationships for structured data management

Validation Rules to ensure data accuracy and consistency

Record-Triggered Flows for process automation

Apex Triggers and Handlers for advanced logic implementation

Reports and Dashboards for real-time performance insights
